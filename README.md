## 🌟 Features

### 🚀 Core Features
- ✅ **Smart Routing** - AI-powered route optimization
- ✅ **Mobile-First** - Runs directly on Android via Termux
- ✅ **No VPS Required** - Fully decentralized architecture
- ✅ **Multi-Language** - Supports Persian, English, and Arabic
- ✅ **Real-Time Tracking** - Live driver and passenger tracking
- ✅ **Payment Integration** - Iranian payment gateways support

### 🎯 Hormozgan-Specific Features
- 🏝️ **Island Services** - Special routes for Qeshm, Kish, Hormoz
- ⚓ **Port Integration** - Seamless connection with Bandar Abbas port
- ✈️ **Airport Services** - Direct airport transfer management
- 🏨 **Tourism Support** - Hotel and resort partnerships

## 🚀 Quick Start

### Prerequisites
- Android device (Android 7+)
- Termux app installed
- Basic terminal knowledge

### Installation

```bash
# Install dependencies
pkg update && pkg upgrade -y
pkg install git nodejs -y

# Clone repository
git clone https://github.com/mahdihadari59-lgtm/hormozgan-driver-pro121.git
cd hormozgan-driver-pro121

# Setup and run
cd cli
npm install
npm start
