# 🇬🇧 English

## 🌟 Features

### 🚀 Core Features
- ✅ **Smart Routing** - AI-powered route optimization with A* algorithm
- ✅ **Mobile-First** - Runs directly on Android via Termux
- ✅ **Real-Time Tracking** - Live location tracking for drivers and passengers
- ✅ **Multi-Language** - Persian, English, and Arabic support
- ✅ **Revenue Management** - Complete income tracking and analytics
- ✅ **Rating System** - 5-star rating for drivers and passengers

### 🎯 Hormozgan-Specific Features
- 🏝️ **Island Services** - Routes for Qeshm, Kish, Hormoz islands
- ⚓ **Port Integration** - Bandar Abbas port connection
- ✈️ **Airport Services** - Direct transfers to/from airport
- 🏨 **Tourism Support** - Hotel and resort partnerships

### 🆕 NEW: Web Application
- 📱 **Driver Dashboard** - Complete web control panel
  - Real-time earnings display
  - Trip history and analytics
  - Profile management
  - Online/Offline status toggle
- 📞 **In-App Call System** - Built-in communication
  - Incoming call handling
  - Call controls (mute, speaker)
  - Call history with details
  - Quick redial
- 🎨 **Modern UI/UX** - Beautiful, responsive design
