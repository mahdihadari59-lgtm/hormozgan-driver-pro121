# 🚕 Hormozgan Driver Pro

پلتفرم هوشمند تاکسی‌یابی استان هرمزگان

## ویژگی‌ها

- 🚗 مدیریت رانندگان و مسافران
- 📊 داشبورد آماری لحظه‌ای
- 🗄️ دیتابیس JSON
- 🎨 رابط کاربری زیبا و ریسپانسیو
- 🔄 به‌روزرسانی خودکار

## نصب و راه‌اندازی

\`\`\`bash
# نصب وابستگی‌ها
npm install

# اجرا
node server.js
\`\`\`

## دسترسی

- Frontend: http://localhost:3000
- API: http://localhost:3000/api/
