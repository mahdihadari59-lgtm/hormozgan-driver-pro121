🌐 پیکربندی دپلوی خودکار

Netlify:
• Build Command: npm run build
• Publish Directory: .
• Environment: NODE_VERSION=18

Vercel:
• به صورت خودکار از vercel.json استفاده می‌کند

Railway:
• به صورت خودکار از railway.json استفاده می‌کند

✨ ویژگی‌های خودکار سازی شده

🔄 پورت خودکار:
• تشخیص خودکار پورت از محیط
• پشتیبانی از Netlify, Vercel, Railway
• fallback به پورت 3000
